# Simple Python Swarm Boilerplate

This is a lightweight, dependency-free (almost) boilerplate for building multi-agent swarms in Python.

## Structure
- `swarm.py`: Contains the core `Agent` and `Swarm` classes.

## Getting Started
1. Install dependencies (optional, but recommended for extension):
   ```bash
   pip install -r requirements.txt
   ```
2. Run the swarm:
   ```bash
   python swarm.py
   ```

## Key Concepts
- **Agent**: A discrete unit with a role and instructions.
- **Swarm**: An orchestrator that manages communication between agents.
- **Workflow**: A sequence of hand-offs or collaborative tasks.
